# 都道府県別人口構成マップ

各都道府県の人口を年代別で折れ線グラフ📈として表示する

## 使用する前に
1. [RESAS API](https://opendata.resas-portal.go.jp/)に登録して`API_KEY`を取得する
2. `.env`ファイルに`REACT_APP_API_KEY=[API_KEY]`と追加する

## 使い方
1. `git clone `
2. `cd population-map`
3. `yarn install`
4. `yarn start`
5. [localhost:3000](http://localhost:3000)を開く
